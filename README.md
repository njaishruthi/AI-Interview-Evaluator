🤖 AI Interview Evaluation System

An intelligent, multimodal interview preparation platform that evaluates candidates using **text, audio, and video inputs**. It provides structured feedback, tracks progress, and motivates users through streak-based learning.

---

🚀 Features

- 📝 **Text Analysis**
  - Evaluates answers based on relevance, clarity, and structure

- 🎤 **Audio Evaluation**
  - Detects response presence (extendable to tone & confidence)

- 🎥 **Video Interaction**
  - Simulates real interview experience using camera input

- 🎯 **HR & Technical Interview Modes**
  - Practice different types of interview questions

- 📊 **Performance Feedback Dashboard**
  - Breakdown of:
    - Text Score
    - Audio Score
    - Video Score
    - Final Score

- 🔥 **Streak Tracking**
  - Encourages consistent daily practice

- 📈 **Progress Tracking**
  - Stores past attempts and visualizes improvement over time

---

🧠 How It Works

1. Select interview type (HR / Technical)
2. Start interview session
3. Answer questions using:
   - Text input
   - Audio recording
   - Camera interaction
4. AI evaluates performance
5. Feedback and scores are generated
6. Progress is stored and visualized

---

🛠️ Tech Stack

- **Frontend & App:** Streamlit  
- **Backend:** Python  
- **Data Handling:** Pandas  
- **AI Models:** Custom evaluation logic (text/audio/video)
